# Predicting-House-Prices-with-Linear-Regression-Machine-Learning-Project-

1. Project Overview

This project builds a Linear Regression model to predict house prices based on property features.

The objective is to:

Understand how different housing attributes influence price

Apply a complete machine learning workflow

Evaluate model performance using regression metrics

The project includes data preprocessing, feature encoding, scaling, model training, and evaluation.

2. Dataset
2.1 File Used

Housing.csv

2.2 Dataset Description

The dataset contains property-related features such as:

area

bedrooms

bathrooms

stories

mainroad

guestroom

basement

hotwaterheating

airconditioning

parking

prefarea

furnishingstatus

price (Target Variable)

4. Project Workflow

The project follows these steps:

Data loading using Pandas

Data inspection and handling missing values

Encoding categorical variables

Feature scaling using StandardScaler

Train-test split (80% training, 20% testing)

Model training using Linear Regression

Performance evaluation using:

R² Score

Mean Absolute Error (MAE)

Visualization of Actual vs Predicted prices

5. Model Evaluation

The model performance is measured using:

R² Score

Indicates how well the model explains price variation

Mean Absolute Error (MAE)

Measures the average prediction error
